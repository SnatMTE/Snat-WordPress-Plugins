== Description ==

Displays a reference via the Harvard standard.

== Installation ==

1) Upload the zip or install it from WordPress.
2) Activate it.
3) Go to the widget and place it where you want. It is called \"Snat\'s Reference Widget\".
4) Done.

== Frequently Asked Questions ==

= Can I change it to another style? =

In future, yes but right now it will only display it as a Harvard reference.
