Displays a reference via the Harvard standard.

== Installation ==
1) Upload the zip or install it from WordPress.
2) Activate it.
3) Go to the widget and place it where you want. It is called \"Snat\'s Reference Widget\".
4) Done.

== Frequently Asked Questions ==
# The name isn\'t showing.

This isn\'t set up to show a username or organisation. Make sure your authors have a first name and last name set up. I might change it in future toa allow you to put an organisation or at least the username.
